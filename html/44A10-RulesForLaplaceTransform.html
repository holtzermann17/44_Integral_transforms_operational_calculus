rules for Laplace transform


If  ℒ⁢{f⁢(t)}=F⁢(s),  then

  • •

    ℒ⁢{ea⁢t⁢f⁢(t)}=F⁢(s-a)  for  s>a,

  • •

    ℒ⁢{f⁢(ta)}=a⁢F⁢(a⁢s)   for  a>0.

For deriving these rules, we start from the definition of Laplace transform.  In the first case, we shall use the notation  s-a=r:

ℒ⁢{ea⁢t⁢f⁢(t)}=∫0∞e-s⁢t⁢ea⁢t⁢f⁢(t)⁢𝑑t=∫0∞e-(s-a)⁢t⁢f⁢(t)⁢𝑑t=∫0∞e-r⁢t⁢f⁢(t)⁢𝑑t=F⁢(r)=F⁢(s-a).

In the second case, we make the change of variable  ta=u  and later use the notation  s⁢a=r:

ℒ⁢{f⁢(ta)}=∫0∞e-s⁢t⁢f⁢(ta)⁢𝑑t=a⁢∫0∞e-s⁢a⁢u⁢f⁢(u)⁢𝑑u=a⁢∫0∞e-r⁢u⁢f⁢(u)⁢𝑑u=a⁢F⁢(r)=a⁢F⁢(a⁢s).
